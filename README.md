# Flask Portfolio Website

A simple personal portfolio website built using Python Flask and HTML templates.

## Features
- Home page with introduction
- Skills list
- Contact form
- Flask routing
- Template rendering

## Project Structure
portfolio/
│── app.py
│── requirements.txt
│── static/
│     └── style.css
└── templates/
      ├── index.html
      └── contact.html

## Run the Project
pip install -r requirements.txt
python app.py

Then open:
http://127.0.0.1:5000/
